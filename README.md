# DocbookCS docs

Documentation site for [DocbookCS](https://github.com/php/docbook-cs), built with [Docusaurus](https://docusaurus.io/).

Published at: https://php.github.io/docbook-cs/

## Local development

```bash
npm install
npm start
```

## Build

```bash
npm run build
```

## Deployment

Deployed automatically to GitHub Pages via GitHub Actions on every push to `main`.
